Enumeration GrpcErrorCode

Standard gRPC status codes as defined in the gRPC specification. These codes provide semantic meaning for different types of errors and enable proper error handling across gRPC services.

See

https://grpc.github.io/grpc/core/md_doc_statuscodes.html

Example

// Throw a specific gRPC error
throw new GrpcException({
  code: GrpcErrorCode.NOT_FOUND,
  message: 'User not found'
});

// Handle errors by code
if (error.code === GrpcErrorCode.UNAUTHENTICATED) {
  // Redirect to login
}
